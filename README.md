## 安装

```bash
# 通过 npm 安装
npm i mone -S

# 通过 yarn 安装
yarn add mone
```

## 快速上手

```js
import Vue from 'vue';
import { Button } from 'mone';
import 'mone/lib/index.css';

Vue.use(Button);
```

mone 也支持按需引入、CDN 引入等方式，详细说明见 [快速上手](https://jczzq.github.io/mone#/zh-CN/quickstart).
